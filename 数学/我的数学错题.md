# 1000题p12 No7

$$
  y(x) = \ln|e^{2x} - 1| ) 的斜渐近线为（） A.( y = 2x + \dfrac{1}{e} ) B. ( y = 2x ) C. ( y = -2x + \dfrac{1}{e} ) D. ( y = -2x )
$$
- 先用$\lim\limits_{x \to +\infty} \frac{\ln (e^{2x} - 1)}{x}$求出极限为2也就是$a = 2$
- 在算$$\lim\limits_{x \to +\infty}f(x) - 2x = \lim_{x \to + \infty}\ln(e^{2x}-1) -2x$$
- 时，是无穷比无穷型，这时用倒代换也做不出来，这里有个[[计算技巧]]即
- 遇到形如$$\ln{e^x} - x$$
- 这种式子，将$x$写成$e^{ln x}$此时对于上面的公式，可以写成
- $$\lim_{x \to \infty}\ln{(e^{2x} - 1) - \ln e^{2x}} = \lim_{x \to +\infty}\ln \frac{e^{2x} - 1}{e^{2x}} = \ln 1 = 0$$
- 所以$a = 2, b = 0$答案选B
- 除了将x改写，还有另一种这个极限的解法
- 在[在对数中提取公因子](计算技巧.md#在对数中提取公因子)中，在$e^{2x - 1}$里提取一个$e^{2x}$原式变成$$\lim_{x \to +\infty}\ln\left(e^{2x}\left(1-e^{-2x}\right)\right) - 2x=\ln e^{2x}+\ln\left(1-e^{-2x}\right) - 2x = \lim_{x \to +\infty}\ln(1-e^{-2x})=0$$

# 1000题

## 基本信息

**题目**：求曲线的渐近线条数

曲线方程：

$$y = e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1}$$

**错题原因**：

1. 求斜渐近线时，未对 $(x \to +\infty)、(x \to -\infty)$ 分类讨论，忽略 $(e^{-\frac{1}{x}})$ 极限差异，以及根号 $(\sqrt{x^2})$ 正负性区别
    
2. 求斜渐近线截距时，极限计算卡顿，不会用洛必达法则化简求解
    

## 解题步骤（完整版）

### 第一步：求定义域，找垂直渐近线

函数有意义需满足：

$$\begin{cases} x \neq 0 \\ x^2 - 4x + 1 \ge 0 \end{cases}$$

解二次不等式：$(x \le 2 - \sqrt{3}) 或(x \ge 2 + \sqrt{3})，间断点为 (x = 0)$。

判断间断点处极限：

$$\lim\limits_{x \to 0^{\pm}} e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1}$$

- $(x \to 0^+) 时，(-\frac{1}{x} \to -\infty)，(e^{-\frac{1}{x}} \to 0)$，极限为0
    
- $(x \to 0^-) 时，(-\frac{1}{x} \to +\infty)，(e^{-\frac{1}{x}} \to +\infty)，整体极限为 (+\infty)$
    

因此，

$$\boldsymbol{x = 0}$$

是**垂直渐近线**。

### 第二步：讨论水平渐近线

#### 1. \(x \to +\infty\)

$$\lim\limits_{x \to +\infty} e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1}$$

先算指数极限：

$$\lim\limits_{x \to +\infty} -\frac{1}{x} = 0$$

故 \(e^{-\frac{1}{x}} \to 1\)。

原式极限：

$$\lim\limits_{x \to +\infty} \sqrt{x^2 - 4x + 1} = +\infty$$

无水平渐近线。

#### 2. \(x \to -\infty\)

$$\lim\limits_{x \to -\infty} e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1}$$

$$\lim\limits_{x \to -\infty} -\frac{1}{x} = 0$$

故 \(e^{-\frac{1}{x}} \to 1\)，根号部分趋于 \(+\infty\)，无水平渐近线。

### 第三步：讨论斜渐近线（核心易错点，分类讨论）

#### 情况一：\(x \to +\infty\)

1. 求斜率 \(k_1\)

$$k_1 = \lim\limits_{x \to +\infty} \frac{e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1}}{x}$$

\(e^{-\frac{1}{x}} \to 1\)，\(x \to +\infty\) 时，\(\sqrt{x^2} = x\)，化简得：

$$k_1 = \lim\limits_{x \to +\infty} \frac{\sqrt{x^2 - 4x + 1}}{x} = \lim\limits_{x \to +\infty} \sqrt{1 - \frac{4}{x} + \frac{1}{x^2}} = 1$$

2. 求截距 \(b_1\)

$$b_1 = \lim\limits_{x \to +\infty} \left( e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1} - x \right)$$

令 \(t = \frac{1}{x}\)，则 \(x \to +\infty\) 时，\(t \to 0^+\)，换元化简，结合洛必达法则，最终求得：

$$b_1 = -3$$

因此，对应的斜渐近线为：

$$\boldsymbol{y = x - 3}$$

#### 情况二：\(x \to -\infty\)

1. 求斜率 \(k_2\)

$$k_2 = \lim\limits_{x \to -\infty} \frac{e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1}}{x}$$

\(e^{-\frac{1}{x}} \to 1\)，\(x \lt 0\) 时，\(\sqrt{x^2} = -x\)，化简得：

$$k_2 = \lim\limits_{x \to -\infty} \frac{\sqrt{x^2 - 4x + 1}}{x} = \lim\limits_{x \to -\infty} \frac{-x \cdot \sqrt{1 - \frac{4}{x} + \frac{1}{x^2}}}{x} = -1$$

2. 求截距 \(b_2\)

$$b_2 = \lim\limits_{x \to -\infty} \left( e^{-\frac{1}{x}} \cdot \sqrt{x^2 - 4x + 1} + x \right)$$

同理换元+洛必达法则，求得：

$$b_2 = 3$$

因此，对应的斜渐近线为：

$$\boldsymbol{y = -x + 3}$$

## 最终结论

垂直渐近线：

$$x = 0$$

斜渐近线：

$$y = x - 3, \quad y = -x + 3$$

总计 **3条** 渐近线。

## 复盘易错点

1. 含 \(\sqrt{x^2}\) 的极限，\(x \to +\infty\) 和 \(x \to -\infty\) 必须分开讨论，正负号极易出错
    
2. 斜渐近线截距计算，遇到无穷减无穷型不定式，优先换元+洛必达法则，避免硬算出错
    
3. 求解渐近线标准顺序：先找垂直渐近线→再判断水平渐近线→最后求斜渐近线，防止漏判
