# 常用积分公式（精简版） 
> 标注 `★` 为高频考点，建议重点记忆 
--- 
## ① 幂函数积分 $$ \int x^k \mathrm{d}x = \frac{1}{k+1}x^{k+1} + C \quad (k \neq -1) $$ $$ \int \frac{1}{x^2} \mathrm{d}x = -\frac{1}{x} + C, \quad \int \frac{1}{\sqrt{x}} \mathrm{d}x = 2\sqrt{x} + C $$ 
---
## ② 对数相关积分 $$ \int \frac{1}{x} \mathrm{d}x = \ln|x| + C \quad (\text{导数：}(\ln x)' = \frac{1}{x}) $$ 
---
## ③ 指数函数积分 $$ \int e^x \mathrm{d}x = e^x + C $$ $$ \int a^x \mathrm{d}x = \frac{a^x}{\ln a} + C \quad (a>0,a\neq1) \quad (\text{导数：}(a^x)' = a^x \ln a) $$ 
---
## ④ 三角函数积分 
### 基础三角积分 $$ \int \sin x \mathrm{d}x = -\cos x + C, \quad \int \cos x \mathrm{d}x = \sin x + C $$ $$ \int \tan x \mathrm{d}x = -\ln|\cos x| + C, \quad \int \cot x \mathrm{d}x = \ln|\sin x| + C $$ 
### 高频考点（★） $$ \int \sec x \mathrm{d}x = \int \frac{\mathrm{d}x}{\cos x} = \ln|\sec x + \tan x| + C $$ $$ \int \csc x \mathrm{d}x = \int \frac{\mathrm{d}x}{\sin x} = \ln|\csc x - \cot x| + C $$ 
### 平方/乘积积分 $$ \int \sec^2 x \mathrm{d}x = \tan x + C, \quad \int \csc^2 x \mathrm{d}x = -\cot x + C $$ $$ \int \sec x \tan x \mathrm{d}x = \sec x + C, \quad \int \csc x \cot x \mathrm{d}x = -\csc x + C $$ 
---
## ⑤ 反正切相关积分 $$ \int \frac{1}{1+x^2} \mathrm{d}x = \arctan x + C $$ $$ \int \frac{1}{a^2+x^2} \mathrm{d}x = \frac{1}{a}\arctan\frac{x}{a} + C \quad (a>0) $$ 
---
## ⑥ 反正弦相关积分 $$ \int \frac{1}{\sqrt{1-x^2}} \mathrm{d}x = \arcsin x + C $$ $$ \int \frac{1}{\sqrt{a^2-x^2}} \mathrm{d}x = \arcsin\frac{x}{a} + C \quad (a>0) $$
--- 
## ⑦ 根式+对数积分（双曲/对数型） $$ \int \frac{1}{\sqrt{x^2+a^2}} \mathrm{d}x = \ln\left(x+\sqrt{x^2+a^2}\right) + C \quad (\text{常见 } a=1) $$ $$ \int \frac{1}{\sqrt{x^2-a^2}} \mathrm{d}x = \ln\left|x+\sqrt{x^2-a^2}\right| + C \quad (|x|>|a|) $$ 
---
## ⑧ 分式型积分（平方差/倒数平方） $$ \int \frac{1}{x^2-a^2} \mathrm{d}x = \frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C $$ $$ \int \frac{1}{a^2-x^2} \mathrm{d}x = \frac{1}{2a}\ln\left|\frac{x+a}{x-a}\right| + C $$ 
> 推导关联：$\int \frac{1}{a^2-x^2} \mathrm{d}x = -\int \frac{1}{x^2-a^2} \mathrm{d}x = -\frac{1}{2a}\ln\left|\frac{x-a}{x+a}\right| + C = \frac{1}{2a}\ln\left|\frac{x+a}{x-a}\right| + C$ 
 
--- 
## ⑨ 根式+反正弦积分（圆型） $$ \int \sqrt{a^2-x^2} \mathrm{d}x = \frac{a^2}{2}\arcsin\frac{x}{a} + \frac{x}{2}\sqrt{a^2-x^2} + C \quad (a>0, |x|\geq0) $$ 
---
## ⑩ 三角平方积分（降幂核心） 
### 正弦/余弦平方 $$ \int \sin^2 x \mathrm{d}x = \frac{x}{2} - \frac{\sin2x}{4} + C \quad \left(\sin^2 x = \frac{1-\cos2x}{2}\right) $$ $$ \int \cos^2 x \mathrm{d}x = \frac{x}{2} + \frac{\sin2x}{4} + C \quad \left(\cos^2 x = \frac{1+\cos2x}{2}\right) $$ 
### 正切/余切平方 $$ \int \tan^2 x \mathrm{d}x = \tan x - x + C \quad \left(\tan^2 x = \sec^2 x - 1\right) $$ $$ \int \cot^2 x \mathrm{d}x = -\cot x - x + C \quad \left(\cot^2 x = \csc^2 x - 1\right) $$
