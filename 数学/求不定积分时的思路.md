# 分母凑平方和
# 📐 二次分母积分：平方和/差配方速记 > 适用场景：$\int \frac{1}{Ax^2+Bx+C}dx$，核心是**配方**化为基本积分公式。 ## 一、判定：能否凑「平方和/差」 
### 核心条件 分母为**二次三项式** $Ax^2+Bx+C$： 
1. 判别式 $\Delta = B^2-4AC$ 
2. **$\Delta < 0$** → 无法因式分解 → 凑**平方和** → 积分结果为 $\arctan$ 
3. **$\Delta > 0$** → 可分解为一次式乘积 → 凑**平方差** → 积分结果为 $\ln$ 分式 
### 速记口诀 
- 二次+不可因式 → 平方和 → $\arctan$ 
- 二次+可因式 → 平方差 → $\ln$ 
## 二、万能配方步骤（通用模板） 对 $Ax^2+Bx+C$ 分两步： 
### 步骤1：提二次系数 $$Ax^2+Bx+C = A\left(x^2+\frac{B}{A}x+\frac{C}{A}\right)$$ ### 步骤2：配方（核心口诀） **一次项系数一半平方，加了再减** $$x^2+px = \left(x+\frac{p}{2}\right)^2-\left(\frac{p}{2}\right)^2$$ ### 步骤3：合并常数 最终化为： $$A\left[(x-h)^2 \pm k^2\right]$$ 
- 「+」：$\Delta < 0$ → 平方和 
- 「-」：$\Delta > 0$ → 平方差 
## 三、三大模板（考研90%） 
### 模板1：纯平方项 $$x^2+a^2$$ 直接套公式： $$\int \frac{1}{x^2+a^2}dx = \frac{1}{a}\arctan\frac{x}{a}+C$$ ### 模板2：含一次项 $$x^2+2bx+c \quad (c>b^2)$$ 配方： $$x^2+2bx+c = (x+b)^2+(c-b^2)$$ 积分： $$\int \frac{1}{(x+b)^2+k^2}dx = \frac{1}{k}\arctan\frac{x+b}{k}+C \quad (k=\sqrt{c-b^2})$$

### 模板3：带二次系数 $$ax^2+bx+c \quad (b^2<4ac)$$ 配方： $$ax^2+bx+c = a\left(x+\frac{b}{2a}\right)^2+\frac{4ac-b^2}{4a}$$ 积分： $$\int \frac{1}{a(u^2+k^2)}dx = \frac{1}{ak}\arctan\frac{u}{k}+C \quad (u=x+\frac{b}{2a}, k=\sqrt{\frac{4ac-b^2}{4a^2}})$$
## 四、实战例题（你的原题） 计算：$2\int \frac{1}{at^2+2bt+a}dt$ 
### 情况1：$a^2 > b^2$（$\Delta < 0$，平方和） 1. 配方： $$at^2+2bt+a = a\left[\left(t+\frac{b}{a}\right)^2+\frac{a^2-b^2}{a^2}\right]$$ 2. 套公式 $\int \frac{1}{u^2+k^2}du = \frac{1}{k}\arctan\frac{u}{k}+C$： $$\frac{2}{a}\int \frac{dt}{\left(t+\frac{b}{a}\right)^2+\left(\frac{\sqrt{a^2-b^2}}{a}\right)^2} = \frac{2}{\sqrt{a^2-b^2}}\arctan\frac{at+b}{\sqrt{a^2-b^2}}+C$$ ### 情况2：$a^2 < b^2$（$\Delta > 0$，平方差） 1. 配方： $$at^2+2bt+a = a\left[\left(t+\frac{b}{a}\right)^2-\frac{b^2-a^2}{a^2}\right]$$ 2. 套公式 $\int \frac{1}{u^2-k^2}du = \frac{1}{2k}\ln\left|\frac{u-k}{u+k}\right|+C$： $$\frac{2}{a}\int \frac{dt}{\left(t+\frac{b}{a}\right)^2-\left(\frac{\sqrt{b^2-a^2}}{a}\right)^2} = \frac{1}{\sqrt{b^2-a^2}}\ln\left|\frac{at+b-\sqrt{b^2-a^2}}{at+b+\sqrt{b^2-a^2}}\right|+C$$
