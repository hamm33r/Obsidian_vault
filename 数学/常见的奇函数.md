### $$f(x) = \ln\frac{1-x}{1+x} \quad (-1<x<1)$$ * **推导**：$f(x) = \ln(1-x) - \ln(1+x)$，显然 $f(-x) = \ln(1+x) - \ln(1-x) = -f(x)$。 
### 1.2 指数构造类 $$f(x) = \frac{1}{2^x+1} - \frac{1}{2} \quad (x\in\mathbb{R})$$ * **变形技巧**：常将复杂指数函数拆分为“奇函数 + 常数”形式。 $$f(x) = g(x) + C \implies \text{若 } g(x) \text{ 为奇函数，则 } f(x) \text{ 的奇偶性由 } g(x) \text{ 决定}.$$ 
### 1.3 三角函数类 $$\sin x, \quad \tan x, \quad \arcsin x, \quad \arctan x$$ * **性质**：最基础的奇偶函数，在高阶导数计算中高频出现。

## 2. 复合函数奇函数  利用**“内偶则偶，内奇同外”**的口诀快速判断复合函数奇偶性。 
### 2.1 三角复合 $$f(x) = \sin(\cos x) + \cos(\sin x)$$ 分析： *$\cos x$ 是偶函数，$\sin(\text{偶函数}) = \text{偶函数}$。 * $\sin x$ 是奇函数，$\cos(\text{奇函数}) = \text{偶函数}$。 * 偶 + 偶 = 偶 $\implies$ 该函数为偶函数。 ### 2.2 抽象复合推论 若 $f(u)$ 为偶函数，$g(x)$ 为奇函数，则： $$f(g(x)) \quad \text{是奇函数}$$原理：内层奇函数将符号带入，外层偶函数保持符号不变，最终结果互为相反数。
