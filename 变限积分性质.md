# 1.函数f(x)在I上可积，则函数$F(x)=\int_{a}^{x}f(t)dt在I上连续$
### $F(x)若存在，则其一定连续$
# 2. 函数f(x)在I上连续，则函数$F(x)=\int_{a}^{x}f(t)dt在I上可导且F'(x)=f(x)$

# 3.若$x=x_0 \in I是f(x)唯一的跳跃间断点,则F(x)=\int_{a}^{x}f(t)dt在x_0处不可导(有尖点)$
# 4.若$x=x_0 \in I是f(x)唯一的可去间断点,则F(x)=\int_{a}^{x}f(t)dt在x_0处可导，$$且F'(x_0)=\lim_{x \to x_0}f(x) \neq f(x_0)$
